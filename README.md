# toolbox-api

Helpers for interfacing with Reddit Moderator Toolbox settings and usernotes.

## Installation

This project is currently very work-in-progress and not yet published anywhere outside Github. Development builds are generated from every branch push via Github Actions; to install the latest build of the `master` branch, use one of the following commands:

```bash
# With Yarn:
$ yarn add --production https://github.com/Geo1088/toolbox-api.git#builds/master
# With npm:
$ npm install --save --production https://github.com/Geo1088/toolbox-api.git#builds/master
```
